# 📖 Sobre o Projeto

O **EletroBot AI** é uma plataforma desenvolvida para automatizar o atendimento de lojas de eletrodomésticos, oferecendo uma experiência inteligente para clientes por meio de canais como WhatsApp e Web.

O sistema utiliza Inteligência Artificial para interpretar perguntas em linguagem natural, consultar o catálogo de produtos, verificar estoque, recomendar itens, criar pedidos e encaminhar o atendimento para um vendedor quando necessário.

O projeto está sendo desenvolvido seguindo padrões modernos de arquitetura de software, com foco em escalabilidade, organização e qualidade do código.

# ⚙ Instalação

Clone o repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

Instale as dependências:

```bash
composer install
```

Configure o arquivo `.env`:

```bash
cp .env.example .env
```

Gere a chave da aplicação:

```bash
php artisan key:generate
```

Execute as migrations:

```bash
php artisan migrate
```

Execute os seeders:

```bash
php artisan db:seed
```

Inicie o servidor:

```bash
php artisan serve
```
