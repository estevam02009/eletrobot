# 🗺️ Roadmap - EletroBot AI

## Visão Geral

O EletroBot AI é uma plataforma inteligente de atendimento para lojas de eletrodomésticos, construída com Laravel 12, Inteligência Artificial e integração com WhatsApp.

Este roadmap descreve a evolução planejada do projeto.

---

# Sprint 0 — Fundação

**Status:** ✅ Concluída

### Objetivos

* Estrutura do projeto
* Configuração do ambiente
* Banco de dados inicial
* Migrations
* Models
* Relacionamentos
* Seeder de categorias
* CRUD de Marcas
* Organização do GitHub
* Documentação inicial
